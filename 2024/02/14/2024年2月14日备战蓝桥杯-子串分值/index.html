2868.子串分值（枚举 贡献法）

https://www.acwing.com/problem/content/2871/

我的思路(暴力超时)

枚举子串，再计算每个子串

计算：用一个数组统计一个字母出现的次数，最后统计数组中元素是1的个数

我的代码

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.time.chrono.JapaneseChronology;


public class Main {

    public static void main(String[] args) throws IOException{
        BufferedReader reader=new BufferedReader(new InputStreamReader(System.in));
        String s=new String(reader.readLine());
        int sum=0;
        for(int i=0;i<s.length();i++) {
            String t="";
            for(int j=i;j<s.length();j++) {
                t+=s.charAt(j);
                sum+=check(t);
            }
        }
        
        System.out.print(sum);

    }

    public static int check(String t) {
        int[] a=new int[30];
        int ans=0;
        for(int i=0;i<t.length();i++) {
            a[t.charAt(i)-'a']+=1;
        }
        for(int i=0;i<30;i++) {
            if(a[i]==1) ans+=1;
        }
        return ans;
    }

}

acwing的思路

不妨考虑每个位置的字母如果能对当前子串做出贡献的话，那么这个子串中就不能再包含与当前位置相同的字母了，从当前字母分别向左和向右找，找到第一个与当前位置相同字母的位置。假设当前位置为j，左边第一个与当前字母相同的位置为i，右边第一个与当前字母相同的位置为k，那么在区间(i+ 1,k- 1)仅包含位置j的子串的个数即为(j-i)*(k-j)个，而每个子串中位置为j的字母都能贡献1，那么总贡献就是(j-i)*(k - j)。最后只要从头到尾算出每个位置做出的贡献相加即可求出答案。

例子：
索引: 1 2 3 4 5
i=1 j=3 k=5
j-i=2 k-j=2 2*2=4
子串组合为: 3 23 34 234 (四个)

边界问题：字符串从索引1开始记录，第一个位置的字母，假设索引0位置的字母与它相同；最后一个位置的字母，假设n+1位置的字母与它相同

acwing的代码

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.math.BigInteger;

public class Main {

    public static void main(String[] args) throws IOException{
        BufferedReader reader=new BufferedReader(new InputStreamReader(System.in));
        String s=new String(reader.readLine());
        String ss=" "+s;//制造从索引1开始记录字符串
        int N=100010;
        int[] l=new int[N];
        int[] r=new int[N];
        int[] p=new int[26];
        
        //找左边字母相同的位置
        for(int i=1;i<ss.length();i++) {
            int t=ss.charAt(i)-'a';
            l[i]=p[t];//记录当前字母左边相同值的位置
            p[t]=i;//更新当前字母的位置
        }
        
        for(int i=0;i<26;i++) p[i]=ss.length();//避免右边界问题，ss.length()==n+1 n为字符串长度，1为开头的空格
        //找右边字母相同的位置
        for(int i=ss.length()-1;i>0;i--) {
            int t=ss.charAt(i)-'a';
            r[i]=p[t];//记录当前字母右边相同值的位置
            p[t]=i;//更新当前字母的位置
        }
        
//		BigInteger sum=BigInteger.ZERO;
//		for(int i=1;i<ss.length();i++) {
//			BigInteger contribution = BigInteger.valueOf(i - l[i]).multiply(BigInteger.valueOf(r[i] - i));
//            sum = sum.add(contribution);  //计算当前字母的贡献值
//		}
        long sum=0;
        for(int i=1;i<ss.length();i++) {
            sum+=(long)(i - l[i])*(r[i]-i);
        }
        
        System.out.print(sum);
    }
}
