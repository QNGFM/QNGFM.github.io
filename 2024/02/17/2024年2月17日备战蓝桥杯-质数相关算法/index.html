质数

质数的定义：只有两个正因数（1和自己）的自然数即为质数。比1大但不是素数的数称为合数。1和0既非素数也非合数。

试除法判定质数

模板
bool is_prime(int x)
{
    if (x < 2) return false;//根据定义
    for (int i = 2; i <= x / i; i ++ )//i <= x / i 即 i*i<=x
        if (x % i == 0)
            return false;
    return true;
}
例题
  1. 试除法判定质数
import java.util.Scanner;

public class Main {

    public static void main(String[] args) {

            long x=scanner.nextLong();

        if(x<2) return false;
        //试除法
        for(long i=2;i<=x/i;i++) {
            if(x%i==0) return false;
        }
        return true;
    }

}

试除法分解质因数

质因数（素因数或质因子）在数论里是指能整除给定正整数的质数。如2整除6，表示为2|6

模板
void divide(int x)
{
    for (int i = 2; i <= x / i; i ++ )
        if (x % i == 0)
        {
            int s = 0;
            while (x % i == 0) x /= i, s ++ ;
            cout << i << ' ' << s << endl;
        }
    if (x > 1) cout << x << ' ' << 1 << endl;
    cout << endl;
}
例题
867. 分解质因数
import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        int n=scanner.nextInt();
        while(n--!=0) {
            long x=scanner.nextLong();
            divide(x);
        }

    }

    public static void divide(long x) {
        for(long i=2;i*i<=x;i++) {
            if(x%i==0) {
                int s=0;
                while(x%i==0) {
                    x/=i;
                    s++;
                }
                System.out.println(i+" "+s);
            }
        }
        if(x>1) System.out.println(x+" "+1);
        System.out.println();
    }

}

求素数

朴素筛法求素数模板

如果没被筛过，则记为素数，然后筛选素数的倍数，如2为素数，把4，6，8，10，12筛掉

以下模板是优化后的，

int primes[N], cnt;     // primes[]存储所有素数
bool st[N];         // st[x]存储x是否被筛掉

void get_primes(int n)
{
    for (int i = 2; i <= n; i ++ )
    {
        if (st[i]) continue;//已经被筛
        primes[cnt ++ ] = i;
        for (int j = i + i; j <= n; j += i)
            st[j] = true;
    }
}
线性筛法求素数模板

数量级为10^6，速度和朴素筛法差不多

数量级为10^7，线性筛法速度快一倍，时间复杂度O(n)

核心：每一个数只会被最小质因子筛掉

primes存的是到i为止的所有质数

通过遍历当前primes来筛掉合数

合数不需要大于n：primes[j] <= n / i 即 primes[j] * i <= n 因为我们只求1到n的素数，超过n的数不用筛

primes[j] * i的最小质因子肯定是primes[j] ，因为primes的质数是从小到大的

当满足i % primes[j] == 0这个条件，我们退出循环，因为此时已找到i的最小质因子，如果不结束循环，不能保证primes[j] * i的最小质因子是primes[j] ，因为i可能包含更小的质因子

int primes[N], cnt;     // primes[]存储所有素数
bool st[N];         // st[x]存储x是否被筛掉

void get_primes(int n)
{
    for (int i = 2; i <= n; i ++ )
    {
        if (!st[i]) primes[cnt ++ ] = i;
        for (int j = 0; primes[j] <= n / i; j ++ )
        {
            st[primes[j] * i] = true;
            if (i % primes[j] == 0) break;
        }
    }
}
例题
868. 筛质数

朴素筛法

import java.util.Scanner;

public class Main {
    public static int N=1000000;
    public static int cnt=0;
    public static int[] primes=new int[N];
    public static boolean[] st=new boolean[N];
    
    public static void getPrimes(int n) {
        for(int i=2;i<=n;i++) {
            if(st[i]) continue;
            primes[cnt++]=i;
            for(int j=i+i;j<=n;j+=i) st[j]=true;
        }
    }

    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        int n=scanner.nextInt();
        getPrimes(n);
        System.out.println(cnt);
        for(int i=0;i<cnt;i++) {
            System.out.print(primes[i]+" ");
        }
        scanner.close();
    }

}

线性筛法

import java.util.Scanner;

public class Main {
    public static int N=1000000;
    public static int cnt=0;
    public static int[] primes=new int[N];
    public static boolean[] st=new boolean[N];
    
    public static void getPrimes(int n) {
        for(int i=2;i<=n;i++) {
            if(!st[i]) primes[cnt++]=i;
            for(int j=0;primes[j]*i<=n;j++) {
                st[primes[j]*i]=true;
                if(i%primes[j]==0) break;
            }
        }
    }

    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        int n=scanner.nextInt();
        getPrimes(n);
        System.out.println(cnt);
        for(int i=0;i<cnt;i++) {
            System.out.print(primes[i]+" ");
        }
        scanner.close();
    }

}
