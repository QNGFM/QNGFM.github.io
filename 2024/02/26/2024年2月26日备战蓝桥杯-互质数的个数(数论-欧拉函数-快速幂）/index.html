4968. 互质数的个数(数论 欧拉函数 快速幂）

https://www.acwing.com/problem/content/4971/

我的思路

概念理解：互质是公约数只有1的两个整数，叫做互质整数。即最大公约数1

（1）两个不同的质数一定是互质数。（2）一个质数，另一个不为它的倍数，这两个数为互质数。

先求出a^b(感觉范围是个问题），遍历1-a^b，通过gcb找到最大公约数是1的x

如果快速幂a^b求模的话。会过滤很多数字吧，这里不懂了

我的代码

暴力超时

import java.util.Scanner;

public class Main {
    static final int MOD=998244353;
    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        long a=scanner.nextLong(),b=scanner.nextLong();
        long n=qmi(a,b);
        long ans=0;
        for(long i=1;i<n;i++) {
            if(gcb(i,n)==1) ans=(ans+1)%MOD;
        }
        System.out.print(ans);
        scanner.close();
    }
    public static int gcb(long a, long b) {
        return (int) (b!=0?gcb(b, a%b):a);
    }
    public static long qmi(long a, long b) {
        long res=1;
        long t=a;
        while(b>0) {
            if((b&1)==1) res=res*t;
            t=t*t;
            b>>=1;
        }
        return res;
    }

}

acwing的思路

欧拉函数

欧拉函数是包括N，题目要求不包括，当N等于1时，根据欧拉函数结果是1

而当a^b==1,题目中x取不到1，所以结果应输出0，所以代码需要判断一下

上面公式写反了

题目要求对答案取模，对a^b取模前是肯定可以整除分母的（因为都是质因子），取模后就不一定能，所以该如何取模呢？

分解a^b,对a^b-1取模就好，后面的单独a能整除完所有p

acwing的代码

import java.util.Scanner;

public class Main {
    static final int MOD=998244353;
    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        long a=scanner.nextLong(),b=scanner.nextLong();
        if(a==1) {
            System.out.print(0);
            return;
        }
        long res=a,x=a;
        for(int i=2;i<=x/i;i++) {
            if(x%i==0) {
                res=res/i*(i-1);//欧拉公式
                while(x%i==0) {//把这个质因子除感觉
                    x/=i;
                }
            }
        }
        if(x>1) res=res/x*(x-1);//确保最后一个质因子
        res=res*qmi(a, b-1)%MOD;//欧拉公式
        System.out.print(res);
        scanner.close();
    }
    
    public static long qmi(long a, long b) {
        long res=1;
        while(b>0) {
            if((b&1)==1) res=res*a%MOD;
            a=a*a%MOD;
            b>>=1;
        }
        return res;
    }

}
