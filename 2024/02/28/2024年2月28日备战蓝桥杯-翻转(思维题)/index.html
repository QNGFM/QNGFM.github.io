4967. 翻转(思维题）

https://www.acwing.com/problem/content/4970/

我的思路

没有数学推测，只是大胆猜测

贪心：遍历对比两组数，如果不同，判断能否翻转，能则次数加1，不能则打印-1

不能反转条件：边界数字或两边数字不相等的数字

我的代码

结果超时，需要将字符串转为数组，代码在下面

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class Main {

    public static void main(String[] args) throws NumberFormatException, IOException {
        BufferedReader reader=new BufferedReader(new InputStreamReader(System.in));
        int n=Integer.parseInt(reader.readLine());
        while(n-->0) {
            long res=0;
            String s1=reader.readLine();
            String s2=reader.readLine();
            int l=s1.length();
            for(int i=0;i<l;i++) {
                if(s1.charAt(i)!=s2.charAt(i)) {
                    if(i!=0&&i!=l-1&&s2.charAt(i-1)!=s2.charAt(i)&&s2.charAt(i-1)==s2.charAt(i+1)) {
                        res++;
                        StringBuilder sb=new StringBuilder(s2);
                        if(s2.charAt(i)-'0'==0) {
                            sb.setCharAt(i, '1');//替换字符串中的字符
                        }
                        else {
                            sb.setCharAt(i, '0');
                        }
                        s2=sb.toString();
                    }
                    else {
                        res=-1;
                        break;
                    }
                }
            }
            System.out.println(res);
        }

    }

}

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class Main {

    public static void main(String[] args) throws NumberFormatException, IOException {
        BufferedReader reader=new BufferedReader(new InputStreamReader(System.in));
        int n=Integer.parseInt(reader.readLine());
        while(n-->0) {
            long res=0;
            String s1=reader.readLine();
            char[] c1=s1.toCharArray();
            String s2=reader.readLine();
            char[] c2=s2.toCharArray();
            int l=s1.length();
            for(int i=0;i<l;i++) {
                if(c2[i]!=c1[i]) {
                    if(i!=0&&i!=l-1&&c2[i-1]==c2[i+1]&&c2[i-1]!=c2[i]) {
                        res++;
                        c2[i]=c1[i];
                    }
                    else {
                        res=-1;
                        break;
                    }
                }
            }
            System.out.println(res);
        }

    }

}

acwing的思路

10^6，时间复杂度控制在O(n)

注意审题，不是随意翻转，0变1，两边得是1；1变0，两边得是0

性质1：00和11都不能操作。假设左边0要变成1，它的右边得是1，但显然矛盾
性质2：操作不可逆。变化之后不能变回来
性质3：操作顺序不影响操作结果。
根据性质3，我们仅须考虑从左到右操作的所有方案即可。

从左到右考虑每个字符：
情况1：两个字符相等，一定不能操作；
情况2：两个字符不相等，则一定要操作；

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class Main {

    public static void main(String[] args) throws NumberFormatException, IOException {

            char[] c1=s1.toCharArray();
            String s2=reader.readLine();
            char[] c2=s2.toCharArray();
            int l=s1.length();
            for(int i=0;i<l;i++) {
                if(c1[i]!=c2[i]) {
                    if(i==0||i==l-1||c2[i-1]==c2[i]||c2[i+1]==c2[i]) {
                        res=-1;
                        break;
                    }
                    res++;
                    c2[i]=c1[i];
                }
            }
            System.out.println(res);
        }

    }
