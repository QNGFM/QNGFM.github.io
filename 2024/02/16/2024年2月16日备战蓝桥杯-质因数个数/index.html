4658. 质因数个数(数论 分解质因数）

我的思路（暴力超时）

先找出比n小的质数，再遍历质数是否是n的约数，由此得出个数（注定暴力超时）

我的代码

import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        int n=scanner.nextInt();
        List<Integer> zhi=new ArrayList<Integer>();//不是引用对象，需要返回
        zhi=findZhi(n,zhi);
        int res=getCount(n, zhi);
        System.out.print(res);
        scanner.close();
    }

    public static int getCount(int n, List<Integer> zhi) {
        int sum=0;
        for(int i=0;i<zhi.size();i++) {
            if(n%zhi.get(i)==0) sum++;
        }
        return sum;
    }

    public static List<Integer> findZhi(int n, List<Integer> zhi) {
        for(int i=2;i<n;i++) {
            int j;
            for(j=2;j*j<=i;j++) {
                if(i%j==0) {
                    break;
                }
            }
            if(j*j>i) {
                zhi.add(i);
            }
        }
        return zhi;
    }

    

}

acwing的思路（试除法）

对于30% 的评测用例，1≤n≤10000。可以一个个枚举

对于所有评测用例，1≤n≤10^16。试除法，根号n,即10^8,2秒内可以算完

2s可以算2亿次，即2*10^8

第一步枚举约数，从2到根号n（因为d是n的约数，n/d也肯定是n的约数，不用枚举完）

第二步得出质因子，每找到一个约数就把他除干净，如果一个数可以这整除n，那么我们把这个数给除干净，后面就不会再出现这个数，例如p可以整除n，p=a1*a2…，如果a1前面枚举过了，是不会再出现的，所以p不会成立

第三步检查是否找完质因子，如果最后n>1的话，会有一个大于1的质因子没有被除去

注意事项：涉及到10^6的变量，用long类型

acwing的代码

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        long n=scanner.nextLong();
        int res=0;
        for(long i=2;i*i<=n;i++) {
            if(n%i==0) {
                while(n%i==0) n/=i;//把i除干净
                res++;
            }
        }
        if(n>1) res++;
        System.out.print(res);
        scanner.close();
    }
}
