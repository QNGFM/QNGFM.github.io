AcWing 1209. 带分数（dfs全排列）

https://www.acwing.com/problem/content/1211/

我的思路：

注意特征：带分数中，数字 1∼9 分别出现且只出现一次（不包含 0）。

根据上面这一句话，可以知道是枚举，但感觉很复杂

我的思路是:先排列型枚举9个数字，然后进行分割验算，切割成三部分(两层循环实现)

验证等式 由 n=a+b/c变成 n*c=a*c+b (除法变乘法好运算)

我的代码：
import java.util.Scanner;

public class Main {
    static int N=15;
    static int n;
    static int[] st=new int[N];
    static boolean[] used=new boolean[N];
    static int res=0;
    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        n=scanner.nextInt();
        
        dfs(1);
        
        System.out.print(res);
        scanner.close();

    }

    public static void dfs(int u) {
        if(u>9) {
            for(int i=1;i<=7;i++) {//留两位给后面的b,c
                for(int j=i+1;j<=8;j++) {//留一位给后面的c
                    int a=getNum(1,i);
                    int b=getNum(i+1,j);
                    int c=getNum(j+1,9);
                    if(check(a,b,c)) res++;
                }
            }
        }
        
        for(int i=1;i<=9;i++) {
            if(!used[i]) {
                st[u]=i;
                used[i]=true;
                dfs(u+1);
                
                //恢复现场
                st[u]=0;
                used[i]=false;
            }
        }
        
    }

    private static boolean check(int a, int b, int c) {
        if(n*c==a*c+b) return true;
        return false;
    }

    private static int getNum(int l, int r) {
        int temp=0;
        for(int i=r;i>=l;i--) {
            temp=temp*10+st[i];
        }
        return temp;
    }
    
    

}

针对a的叶子节点再去枚举c

判断b：根据a,c求出b，再逐位标记b中的数字，最后检查所有数字都已经用上

acwing的代码:
import java.util.Scanner;

public class Main {
    static int N=15;
    static int n;
    static int[] st=new int[N];
    static boolean[] used=new boolean[N];
    static int res=0;
    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        n=scanner.nextInt();
        
        dfs_a(0,0);
        
        System.out.print(res);
        scanner.close();

    }

    public static void dfs_a(int u,int a) {//u表示枚举的位数，
        if(a>=n) return;//根据等式剪枝
        if(a!=0) dfs_c(u,a,0);
        
        for(int i=1;i<=9;i++) {
            if(!used[i]) {
                used[i]=true;
                dfs_a(u+1,a*10+i);
                
                //恢复现场
                used[i]=false;
            }
        }
        
    }

    private static void dfs_c(int u, int a, int c) {
        if(u>=9) return;//剪枝，已经没有数给b
        
        if(check(a, c)) res++;
        
        for(int i=1;i<=9;i++) {
            if(!used[i]) {
                used[i]=true;
                dfs_c(u+1,a,c*10+i);
                
                //恢复现场
                used[i]=false;
            }
        }
        
        
    }

    private static boolean check(int a, int c) {
        long b=(long)n*c-a*c;
        if(a==0||b==0||c==0) return false;//不允许有0出现
        boolean[] backup=used.clone();
        
        while(b!=0) {
            int x=(int) (b%10);
            b/=10;
            if(x==0||backup[x]) return false;//为0或之前用过了
            backup[x]=true;//标记使用了
        }
        
        for(int i=1;i<=9;i++) {
            if(!backup[i]) return false;//还有数字没填上
        }
        
        return true;
    }


}
