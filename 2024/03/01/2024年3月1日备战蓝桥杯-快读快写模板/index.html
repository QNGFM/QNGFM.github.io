快读模板

单独数字快读

static class FastReader{
        StreamTokenizer st=new StreamTokenizer(new BufferedReader(new InputStreamReader(System.in)));
        //1快读int类型
        public int nextInt() throws Exception{
            st.nextToken();
            return (int)st.nval;
        }
        //2快读long类型
        public Long nextLong() throws Exception{
            st.nextToken();
            return (long) st.nval;
        }
}

快读快写组合

import java.io.*;
import java.util.StringTokenizer;

public class Main {

    public static void main(String[] args) throws IOException {
        FastReader reader = new FastReader();
        BufferedWriter writer = new BufferedWriter(new OutputStreamWriter(System.out));

        int n = reader.nextInt();
        int m = reader.nextInt();

        writer.write(Integer.toString(n));
        writer.write('\n');

        writer.flush();//最后输出
    }

    static class FastRead {
        StreamTokenizer st = new StreamTokenizer(new BufferedReader(new InputStreamReader(System.in)));

        public int nextInt() throws Exception {
            st.nextToken();
            return (int) st.nval;
        }
    }
}

StringTokenizer

StringTokenizer 是 Java 中的一个类，用于将字符串分割成多个标记（tokens）。它是在早期版本的 Java 中引入的.

st.nextToken() 是 StringTokenizer 类的方法，它用于获取下一个标记（token）。标记是通过指定的分隔符对输入字符串进行分割而得到的子字符串。在这里，st 是 StringTokenizer 的实例。

st.nval 是 StringTokenizer 类的字段，表示最后一次通过 nextToken() 方法获取的标记（token）对应的数值。由于 nextToken() 方法返回的是字符串，nval 被设计为 double 类型，用于存储可能是数字的标记的数值。在这里，通过 (long) st.nval 将 double 类型的数值转换为 long 类型，以符合返回值的类型。

所以，st.nextToken(); return (long) st.nval; 的作用是获取下一个标记并将其转换为 long 类型的数值，用于处理从输入流中读取的数据。
