例题

AcWing 92. 递归实现指数型枚举

每一个数都有两个分支，选或者不选

import java.util.Scanner;

public class Main {
    static final int N=16;
    static int n;
    static int[] st=new int[N];//记录状态，0表示没有考虑，1表示选，2表示不选
    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        n=scanner.nextInt();
        
        dfs(1);
        
        scanner.close();

    }
    public static void dfs(int u) {
        //边界条件
        if(u>n) {//已经枚举完所有数
            for(int i=1;i<=n;i++) {
                if(st[i]==1) {
                    System.out.print(i+" ");
                }
            }
            System.out.println();
            return;
        }
        
        //第一个分支：选
        st[u]=1;
        dfs(u+1);
        st[u]=0;//恢复现场
        
        //第一个分支：不选
        st[u]=2;
        dfs(u+1);
        st[u]=0;//恢复现场
        
    }

}
AcWing 94. 递归实现排列型枚举

估计时间复杂度,得出相应方法=>dfs

按顺序枚举，能保证字典序最小

时间复杂度分析与证明（放缩法）看的朦朦的

import java.io.BufferedWriter;
import java.io.IOException;
import java.io.OutputStreamWriter;
import java.util.Scanner;

public class Main {
    static int N=15;
    static int n;
    static BufferedWriter writer=new BufferedWriter(new OutputStreamWriter(System.out));
    static int[] st=new int[N];//0表示未放任何数
    static boolean[] used=new boolean[N];//false表示未被使用
    public static void main(String[] args) throws Exception{
        Scanner scanner=new Scanner(System.in);
        n=scanner.nextInt();
        
        dfs(1);
        
        scanner.close();
        writer.flush();
    }
    public static void dfs(int u) throws Exception {
        //边界条件
        if(u>n) {
            for(int i=1;i<=n;i++) {
                writer.write(st[i]+" ");
            }
            writer.write("\n");
            return;
        }
        
        //枚举每一种方案
        for(int i=1;i<=n;i++) {
            if(!used[i]) {
                st[u]=i;
                used[i]=true;
                dfs(u+1);
                
                //恢复现场
                st[u]=0;
                used[i]=false;
            }
        }
        
    }

}

AcWing 93. 递归实现组合型枚举
我的思路：

枚举每个位置的数字，思路同94

但是题目要求按照从小到大的顺序输出所有方案，所以需要加限制条件

当前位置枚举的数要大于上一位置的数

acwing的思路：

调用参数加入start，表示从这个数开始枚举，由此确保比前一个位置数大

并且进行剪枝：如果后面的数不足于达到我要输出的数量则结束

u表示当前位置还没选上数，u+n-start<m 这条式子成立说明后面的数不足了

优化完时间快3倍

acwing的代码:
import java.io.BufferedWriter;
import java.io.IOException;
import java.io.OutputStreamWriter;
import java.util.Scanner;

public class Main {
    static int N=30;
    static int n;
    static int m;
    static BufferedWriter writer=new BufferedWriter(new OutputStreamWriter(System.out));
    static int[] st=new int[N];//0表示未放任何数
    static boolean[] used=new boolean[N];//false表示未被使用
    public static void main(String[] args) throws Exception{
        Scanner scanner=new Scanner(System.in);
        n=scanner.nextInt();
        m=scanner.nextInt();
        
        dfs(1,1);
        
        scanner.close();
        writer.flush();
    }
    public static void dfs(int u,int start) throws Exception {
        if(u+n-start<m) return; //剪枝
        //边界条件
        if(u>m) {
            for(int i=1;i<=m;i++) {
                writer.write(st[i]+" ");
            }
            writer.write("\n");
            return;
        }
        
        //枚举每一种方案
        for(int i=start;i<=n;i++) {
            if(!used[i]) {
                st[u]=i;
                used[i]=true;
                dfs(u+1,i+1);
                
                //恢复现场
                st[u]=0;
                used[i]=false;
            }
        }
        
    }

}
