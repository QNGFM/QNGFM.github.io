递推

基本思想

递归是总问题向下分解为同类子问题；递推是先计算子问题，再根据子问题推出推出总问题

例题

AcWing 717. 简单斐波那契
import java.io.BufferedWriter;
import java.io.IOException;
import java.io.OutputStreamWriter;
import java.util.Scanner;

public class Main {

    public static void main(String[] args) throws Exception {
        Scanner scanner=new Scanner(System.in);
        BufferedWriter writer=new BufferedWriter(new OutputStreamWriter(System.out));
        int n=scanner.nextInt();
        int[] f=new int[n+5];
        f[0]=0;f[1]=1;
        for(int i=2;i<n;i++) {
            f[i]=f[i-1]+f[i-2];
        }
        for(int i=0;i<n;i++) {
            writer.write(f[i]+" ");
        }
        
        writer.flush();
        scanner.close();
        
    }

}

AcWing 95. 费解的开关
acwing的思路

核心性质：1.顺序可以任意，最终结果都一样；2.每个格子只能按一次，因为按2次相当于没按

操作完第一行，第二行的操作是固定的，即第一行亮的下面位置是肯定不能按的，第一行不亮的下面位置肯定是要按的

以此类推，下一行肯定取决于上一行

每一行开关的操作，完全被前一行的灯的亮灭状态所决定

最后再判断最后一行的亮灯状态来确定这个方案是否合理

问题1：如何确定第一行的状态，这里的方案是枚举，那怎么枚举呢？

第一行有5个数，可以用二进制表示，1表示操作，0表示不操作，那么总共方案就是0-2^5-1=31，总32种

这样子第一行的状态就确定了，然后我们可以根据移位来判断哪个位置为1，i>>k&1 这个等式如果等于1，那么就表示第k位是1

问题2：怎么进行开关操作

定义偏移量，来确定5个格子 x=[0,0,0,1,-1] y=[0,1,-1,0,0]

acwing的代码
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.util.Arrays;


public class Main {
    static int N=6;//字符串还要留一位装'\0'
    static char[][] g=new char[N][N];
    static char[][] backup=new char[N][N];
    static int[] dx= {0,0,0,1,-1};
    static int[] dy= {0,1,-1,0,0};
    public static void main(String[] args) throws Exception {
        BufferedReader reader=new BufferedReader(new InputStreamReader(System.in));
        int n=Integer.parseInt(reader.readLine());
        
        while(n-->0) {
            for(int i=0;i<5;i++) {//输入初始状态
                g[i]=reader.readLine().toCharArray();
            }
            reader.readLine();//d
            
            int res=10;
            
            for(int i=0;i<g.length;i++) {
                backup[i]=Arrays.copyOf(g[i], g[i].length);//保留一份，便于后续恢复
            }
            
            
            for(int op=0;op<32;op++) {//第一行有32种可能方案
                int step=0;
                for(int i=0;i<5;i++) {//根据op确定第一行状态
                    if(((op>>i)&1)==1) {//1表示操作
                        step++;
                        turn(0,i);//发送操作位置
                    }
                }
                
                for(int i=0;i<4;i++) {//操作后四行
                    for(int j=0;j<5;j++) {
                        if(g[i][j]=='0') {//灯不亮
                            step++;
                            turn(i+1,j);//前一行状态决定下一行
                        }
                    }
                }
                
                boolean dark=false;//检查最后一行，一开始假设都是亮
                for(int i=0;i<5;i++) {
                    if(g[4][i]=='0') {
                        dark=true;
                        break;
                    }
                }
                
                if(!dark) res=Math.min(res, step);
                for(int i=0;i<backup.length;i++) {
                    g[i]=Arrays.copyOf(backup[i], backup[i].length);//保留一份，便于后续恢复
                }
                
                
            }
            
            if(res>6) res=-1;
            System.out.println(res);
        }

    }
    private static void turn(int x, int y) {
        for(int i=0;i<5;i++) {
            int a=x+dx[i],b=y+dy[i];
            //检查边界
            if(a<0||a>=5||b<0||b>=5) continue;
            g[a][b]^=1;//异或1，0变1，1变0
        }
    }
    

}
