503. 借教室(二分 差分）

https://www.acwing.com/problem/content/505/

我的思路

使用差分来记录借教室的情况

每完成一个订单，遍历差分数组进行前缀和计算，看哪天教室数量小于0，这时便可得出答案

我的代码

超时，O(mn) 10^12>10^8

import java.util.Scanner;

public class Main {
    static final int N=(int) (1e6+5);
    static int[] b=new int[N];
    public static void main(String[] args) {
        Scanner scanner=new Scanner(System.in);
        int n=scanner.nextInt(),m=scanner.nextInt();
        int[] kj=new int[n+5];
        for(int i=1;i<=n;i++) {
            kj[i]=scanner.nextInt();
        }
        for(int i=1;i<=n;i++) {
            insert(i, i, kj[i]);
        }
        boolean scs=true;
        int res=0;
        for(int k=1;k<=m;k++) {
            int d=scanner.nextInt(),s=scanner.nextInt(),t=scanner.nextInt();
            if(scs) {
                insert(s, t, -d);
                int tt=0;
                boolean ctn=true;
                for(int i=1;i<=n;i++) {
                    tt+=b[i];
                    if(tt<0) {
                        ctn=false;
                        break;
                    }
                }
                if(!ctn) {
                    scs=false;
                    res=k;
                }
            }
            
        }
        if(scs) {
            System.out.print(res);
        }
        else {
            System.out.println(-1);
            System.out.print(res);
        }
        scanner.close();
    }
    
    public static void insert(int l,int r,int c) {
        b[l]+=c;
        b[r+1]-=c;
    }

}

acwing的思路

![image-20240305000127404](../2024年3月5日备战蓝桥杯-借教室(二分 差分).assets/image-20240305000127404.png)

二分的思想：可用教师数量一定单调下降，刚好二分出最后一天没出现负值的编号

如果这个编号等于最后一个编号，则所有订单通过，否则编号加1即第一个无法满足的订单的编号

acwing的代码

一定要快读，因为会有很多数据输入

b数组一定要开long才够存

import java.io.*;

public class Main {
    static final int N=(int) (1e6+5);
    static int[] d=new int[N];
    static int[] l=new int[N];
    static int[] r=new int[N];
    static int[] w=new int[N];
    static int n,m;
    public static void main(String[] args) throws Exception {
        FastRead reader=new FastRead();
        n=reader.nextInt();m=reader.nextInt();
        for(int i=1;i<=n;i++) {
            w[i]=reader.nextInt();
        }
        for(int i=1;i<=m;i++) {
            d[i]=reader.nextInt();
            l[i]=reader.nextInt();
            r[i]=reader.nextInt();
        }
        int l=0, r=m;
        while(l<r) {
            int mid=l+r+1>>1;
            if(check(mid)) l=mid;
            else r=mid-1;
        }
        if(l==m) System.out.print(0);
        else {
            System.out.println(-1);
            System.out.print(r+1);
        }
    }
    
    public static boolean check(int mid) {
        long[] b=new long[N];
        //构建截止到当前单号的差分数组(需要的教室数量)
        for(int i=1;i<=mid;i++) {
            b[l[i]]+=d[i];
            b[r[i]+1]-=d[i];
        }
        for(int i=1;i<=n;i++) {
            b[i]+=b[i-1];
            if(b[i]>w[i]) return false;//订单要求教室大于已有的
        }
        return true;
    }
    
    static class FastRead {
        StreamTokenizer st = new StreamTokenizer(new BufferedReader(new InputStreamReader(System.in)));

        public int nextInt() throws Exception {
            st.nextToken();
            return (int) st.nval;
        }
    }

}
